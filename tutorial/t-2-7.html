条件分岐 if else

条件によって処理を分岐するにはif文を使います。

if (条件１) {
  条件１が真(true)のときに実行されるブロック
} else {
  条件１が偽(false)のときに実行されるブロック
}

条件１には該当しないけど、条件２に該当する場合、を表すには else if を使います。

if (条件１) {
  条件１が真(true)のときに実行されるブロック
} else if (条件２) {
  条件１は偽(false)だったけど、条件２は真(true)だった場合に実行されるブロック
} else {
  条件１が偽(false)で、条件２も偽(false)だった場合に実行されるブロック
}

条件の書き方

条件は真偽値になるようにします

大小比較

var a = 10;

// より大きい
if (a > 10) {
  // 実行されない
}

// 以上
if (a >= 10) {
  // 実行される
}

// より小さい
if (a < 10) {
  // 実行されない
}

// 以下
if (a <= 10) {
  // 実行される
}

// 等しい
if (a === 10) {
  // 実行される
}

// 等しくない
if (a !== 10) {
  // 実行されない
}
