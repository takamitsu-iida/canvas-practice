演算子

足したり、引いたり、掛けたり、割ったり。

+

2つの数字を加えるか、2つの文字列を結合します。

6 + 9;
'Hello ' + 'world!';
'Hello' + ' ' + 'world';

- * /

数学の計算をする記号です。

9 - 3;
8 * 2; // JavaScript で乗算はアスタリスク
9 / 3;

=

代入です。変数に値を割り当てます。

等しい、ではありません。同じかどうかの判別は別の記号を使います。

var myName = '飯田';

===

等価の判定をします。2つの値と型が互いに等しいかどうかを調べ、true/falseの結果を返します。イコール記号が3個です。 

var myValue = 3;
console.log(myValue === 3);
console.log(myValue === 4);

!===

非等価の判定をします。

var myValue = 3;
console.log(myValue !== 3);
console.log(myValue !== 4);

練習問題

適切な演算子を書いてください。
