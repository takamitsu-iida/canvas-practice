クラス

クラスはオブジェクトを作るための設計図です。

オブジェクトの中で使う値は違うけど、動作としては同じオブジェクトをたくさん作るときにクラスを使います。

たとえば四角形（Rectangle）を考えたとき、幅（width）と高さ（height）の情報が必要です。

いままで学習してきた知識を総動員して幅10、高さ10の四角形を表すオブジェクトを作るなら、こうなります。 

var rect = {};  // 空っぽのオブジェクトを作る

rect.width = 10;  // 幅を10
rect.height = 10;  // 高さを10

さらに、自分自身の面積を計算するcalcArea()という関数をオブジェクトに持たせるなら、こうなります。 

rect.caclArea = function() {
  return this.width * this.height;
}

もしくは、初期値としてこう書いてもいいでしょう。

var rect = {
  width: 10,
  height: 10,
  caclArea: function () {
    return this.width * this.height;
  },
};

四角形を表すこのオブジェクトを一つしか作らないなら、この書き方で十分です。

幅や高さを変えた四角形のオブジェクトをたくさん作る必要があるなら、クラスの出番です。

クラス宣言

クラスはオブジェクトを作るための設計図です。

constructor()という特別な関数の中で、どんな値をもたせたいかを設計します。 

class Rectangle {
  constructor(width, height) {
    this.width = width;
    this.height = height;
  }
}

thisは「自分自身」という特別な意味を持っています。

this.width = width; は自分自身の中にwidthというキーを作り、値として引数で与えられたwidthを代入します。

this.height = height; は自分自身の中にheightというキーを作り、値として引数で与えられたheightを代入します。

これは設計図を宣言しただけなので、実際に使うにはオブジェクトを作る必要があります。

幅10、高さ10の四角形のオブジェクトは次のようにすることで作り出せます。 

var rect = new Rectangle(10, 10);

幅20、高さ20の別の四角形のオブジェクトは、次のように作ります。

var rect2 = new Rectangle(20, 20);

このように、classでオブジェクトの設計図を作り、newで作り出します。
