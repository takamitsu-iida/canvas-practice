変わった演算子の書き方

プログラムを書いていると変数の値を1増やすとか、1減らす、という操作はあまりにも頻出するので、簡単に書けるようになっています。

++と--

変数の値を1増やすには、このように書きます。

var i = 0;
i = i + 1;

こう書いても同じです。より短く書けます。

var i = 0;
i++;

減らす場合も同じです。

var i = 10;
i = i - 1;

こう書いても同じです。より短く書けます。

var i = 10;
i--;

+= と -=

増やしたり減らしたりするのが1ではない場合も多いです。その場合は++ではなく+=を使います。マイナスの場合は-=を使います。

これは変数を2増やす書き方ですが、

var i = 1;
i = i + 2;

このように書くこともできます。

var i = 1;
i += 2;
