等しいかどうかの判定（ == と === ）

==は比較対象を同じ型に変換してから中身を比較します

===は型と値が同じかどうかを判定します。

等しいかどうかの判定に==を使ってもプログラムは動きますが、必ず===を使うようにしましょう。 

var a = 51;
var b = '51';

if (a == b) {
  console.log("==で判定すると、aとbは同じ");
} else {
  console.log("==で判定すると、aとbは同じではない");
}

if (a === b) {
  console.log("===で判定すると、aとbは同じ");
} else {
  console.log("===で判定すると、aとbは同じではない");
}

true, falseの省略

if文の()の中には真偽値（trueかfalse）を書きます。

変数の中身が0や未定義状態だとfalseとして判定されます。 これをうまく使って「変数に中身があるなら」というのを次のように書きます。 

if (a) {
  // aに中身が入っているなら
}

!をつけると真偽値を反転します。

if (!a) {
  // aがfalse、すなわちaの中身が0とか空っぽなら、
}

練習問題

変数aにはtrueかfalseがランダムに入っています。aがtrueの時には「真です」を、falseの時には「偽です」を表示してください。

ブラウザのF5キーを何度か叩いて結果を確認してください。
