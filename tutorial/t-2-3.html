変数

変数を使うには、最初に宣言が必要です。 変数の名前には、アルファベットと記号、数字が使えます。 

var myName;
var a;
var a2;
var hasDone;
var isStarted;

ただし、先頭の文字はアルファベットでなければいけません。

ダメな例

var 2a;

JavaScriptの中で予約されている単語も使えません。

ダメな例

var console;
var Math;
var document;
var false;
var true;

宣言した変数に値を入れるには、イコール記号を使います。

aという変数に1を代入するにはこのようにします。

var a;
a = 1;

イコール記号を使っていますが、aは1と等しい、ではありません。aに1を代入、です。

宣言と同時に値を入れる書き方もあります。この書き方もよく使われます。

var a = 1;

var と let と const

最新のJavaScriptの仕様では、varで宣言するのではなく、letとconstを使うように推奨されています。

letで宣言した変数は特定された範囲内でしか使えないように制限されています。

constは定数で、変更できません。変更してはいけないモノに対して使います。

letとconstは制限が強いですが、そのおかげでバグの少ないプログラムを書けるようになっています。
