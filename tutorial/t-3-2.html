関数の中で使う変数

関数の中で宣言した変数は、その関数の中でしか使えません。

関数の外側でvarで宣言された変数は、関数の中でも使えます。

下記の例では、関数の外側で宣言したcountを関数countUp()の中で操作しています。 

var count = 0;

function countUp() {
  count++;
}

countUp();
console.log(count);  // 1

countUp();
console.log(count);  // 2

このプログラムは動きますが、あまり良いプログラムではありません。

var count = 0;の宣言と、関数の宣言が近い場所にあるので見やすいですが、 遠く離れた場所で宣言されていると、「countってどっから出てきた？」となってしまいます。

このようなどこからでもアクセスできてしまう変数を「グローバル変数」と言います。

グローバル変数はできるだけ使わないようにするのが、バグの少ない、よいプログラムです。

ですが、使わざるを得ない場面も多々あります。

問題

グローバル変数として定義されている幅（width）と高さ（height）を使って 四角形（Rectangle）の面積を求めるcalcRectangleArea関数を作成してください。
