オブジェクト

JavaScriptの全てはオブジェクトです。

今は意味が分からないかもしれませんが、このことはJavaScriptを学習する上で重要なことですので、覚えておいてください。

オブジェクトというのは、関連のあるデータと関数をひとかたまりにしたものです。

オブジェクトを使うことで、分かりやすく読みやすいプログラムが書けるようになります。

オブジェクトは{}で表します。

userという変数を作成して、空っぽのオブジェクトを代入するには、このようにします。 

var user = {};

このオブジェクトにデータを追加するには、このようにします。

user.firstName = "Takamitsu";

userオブジェクトの中では、firstNameというキーに対して、"Takamitsu"という値（文字列）が割り当てられました。

キーは重複しなければいくつでも定義できます。

user.lastName = "Iida";

これでuserオブジェクトの中には２つのキー、firstNameとlastNameが追加されました。

値を変更したい場合は、そのキーに再度代入するだけです。

user.firstName = "Mayo";

これで、先程まで"Takamitsu"だったfirstNameキーが"Mayo"に置き換わりました。

この状態でコンソールにこのuserオブジェクトを出力してみます。

console.log(user);

すると {firstName: 'Mayo', lastName: 'Iida'} と表示されます。

先程は空っぽのオブジェクトを作って、後からキーと値を追加していきましたが、変数を定義するときに完成したオブジェクトを代入することもできます。 

var user = {
  firstName: "Takamitsu",
  lastName: "Iida",
};

キー:値

という書式になっています。

値の後ろにコンマ,が付いていることに注意してください。

練習問題

nameキーに、値"apple"を代入してください。

weightキーに、値150を代入してください。
