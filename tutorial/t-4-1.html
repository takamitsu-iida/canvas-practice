配列

配列は複数のオブジェクトを順序を保って保存する入れ物です。

var names = ['Smith', 'Johnson', 'Williams'];

先頭の要素を取り出すには[0]、2番目の要素を取り出すには[1]、というように数字で指定します。 この数字のことをインデックスといいます。 

console.log(names[0]);  // 'Smith'
console.log(names[1]);  // 'Johnson'
console.log(names[2]);  // 'Williams'

インデックスは0で始まることに注意してください。 したがって、中身が3個の場合、最後の要素のインデックスは2になります。

最後の要素のインデックスは「配列の長さ - 1」になる、ということを覚えておいてください。

配列の長さはlengthというプロパティで取得できます。

console.log(names.length);  // 3

練習問題

要素の数が分からない配列numbersがあります。

配列の長さを調べて、表示してください。

配列の最後に何が入っているか、表示してください。
