関数とは

関数とは、処理の集まりを一つにまとめたものです。 与えられた入力値（引数：ひきすう）に基づいて処理を実行し、結果を返します。 関数を使うことで同じ処理を何度も書かなくて済み、プログラムの見通しがよくなります。

JavaScriptには標準で組み込まれている関数がいくつもあります。 それらを使いつつ、自分で関数を作ってプログラムを完成させていきます。

関数の作り方

function宣言で関数を作ります。

引数は英語でargumentと言いますので、argで表現されることが多いです。

function 関数名 (arg1, arg2) {
  // 処理を
  return 戻り値;
}

例として、底辺と高さを引数としてとり、三角形の面積を計算して返す関数を考えてみます。

関数名は三角形（triangle）の面積(area)を計算(calculate)するので、calcTriangleAreaとします。

引数は２つあり、一つは底辺なのでbottomという名前にします。もう一つは高さなのでheightとします。

関数の定義としてはこのようになります。関数を宣言する最後の中括弧にセミコロン;は書きません。 

function calcTriangleArea(bottom, height) {

}

底辺X高さ÷2を計算してreturnで戻しますので、完成した関数はこうなります。 

function calcTriangleArea(bottom, height) {
  var area = bottom * height / 2;
  return area;
}

この関数が宣言されていれば、この関数を呼び出すことでいつでも三角形の面積を計算できます。 

var b = 10;
var h = 10;

var area = calcTriangleArea(b, h);

console.log("面積は" + area + "です");

問題

四角形（Rectangle）の面積を求めるcalcRectangleArea関数を作成してください。

引数はwidth（長さ）とheight（高さ）としてください。
